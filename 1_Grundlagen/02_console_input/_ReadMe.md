# Consolen Eingabe 

- Fügen Sie der Arbeitsmappe ein neues Projekt hinzu.
- Deklarieren Sie zwei string Variablen für vorname und nachname.
- Lassen Sie den Anwender seinen Vor- und Nachnamen eingeben.
- Zeigen Sie die Meldung "Guten Tag" + vorname + " " + Nachname an 
- Kommentieren Sie im Quellcode alles, was für Sie neu war.
