# Consolen Eingabe 

- Fügen Sie der Arbeitsmappe ein neues Projekt hinzu.
- Deklarieren Sie eine int Variable für eine Schulnute.
- Lassen Sie den Anwender eine Schulnote eingeben.
- Aufgabe 1: Geben Sie mit Hilfe von if ... else if ... für jede Schulnote deren Beschreibung aus.
- Aufgabe 2: Geben Sie mit Hilfe von switch für jede Schulnote deren Beschreibung aus.
- Aufgabe 3: Geben Sie mit Hilfe eines Arrays für jede Schulnote deren Beschreibung aus.
